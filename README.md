## 📖 Descripción

**ArtSpace** es un portafolio interactivo para una artista visual. Este proyecto demuestra habilidades avanzadas de **JavaScript** (filtros dinámicos, lightbox, galería dinámica) y un diseño altamente estético con **CSS Grid avanzado**, tipografías elegantes y microinteracciones.

### ✨ Características destacadas

| Característica | Descripción |
|----------------|-------------|
| 🖼️ **Galería dinámica** | Generada desde un array de objetos (simula API) |
| 🔍 **Filtros por categoría** | Pintura, digital, escultura - actualización instantánea |
| 💡 **Lightbox personalizado** | Visualización ampliada de imágenes sin librerías |
| 📱 **Diseño responsive** | Grid adaptativo, menú hamburguesa |
| 🎨 **Microinteracciones** | Hover con overlay, animaciones suaves |
| 🧭 **Navegación suave** | Smooth scroll entre secciones |
| ✉️ **Formulario de contacto** | Validación y feedback visual |

## 🎯 Lo que aprendí con este proyecto

Este proyecto me llevó al siguiente nivel en JavaScript:

### JavaScript intermedio/avanzado
- **Renderizado dinámico** de elementos HTML desde un array de objetos
- **Filtrado en tiempo real** sin recargar la página
- **Lightbox casero** (manejo de eventos, mostrar/ocultar, control de scroll)
- **Manejo de clases CSS** con JavaScript para menús y filtros activos

### CSS profesional
- **Grid avanzado** con `grid-auto-flow: dense` para layouts asimétricos
- **Overlays al hover** con transformaciones suaves
- **Parallax effect** en el hero (background-attachment)
- **Diseño consistente** con sistema de variables

### UX/UI
- **Feedback visual** en cada interacción (hover, clic, filtro activo)
- **Carga lazy** de imágenes (`loading="lazy"`)
- **Formulario con validación** y mensaje de éxito temporal

## 🧠 ¿Qué hace especial a ArtSpace?

A diferencia de una galería estática, ArtSpace es **completamente dinámica**:

1. **La galería se genera desde JavaScript** - lo que facilita agregar nuevas obras sin tocar HTML.
2. **Los filtros son instantáneos** - muestra solo lo que el usuario quiere ver.
3. **El lightbox es 100% casero** - demuestra que entiendo el flujo de eventos y manipulación del DOM.
4. **El diseño es único** - combinación de tipografías serif y sans, colores tierra, bordes redondeados.
